# **Этап 2 Создание новых признаков (1.5 балла)**

Иногда на основании имеющихся признаков можно создать новые. Например, обработав столбец с датой, можно выяснить: является ли день рабочим, является ли месяц концом квартала, является ли день национальным праздником и т.д. Создайте новые признаки, которые логически вписываются в Ваш набор данных.
Кол-во создаваемых признаков – минимум 2, больше лучше.

# **Этап 3 Разведывательный анализ данных (2.5 балла)**

Применяя методы с семинаров (например, groupby/фильтры по какому-нибудь условию/describe), получите больше информации о данных. Постройте корреляционную матрицу, попробуйте найти зависимости. Чем более интересные находки Вы сделаете, тем более высокий балл за этот шаг получите.

# **Этап 4 Визуализация (2.5 балла)**

Ваша задача – построить красивые и интересные визуализации, на основе которых можно выдвинуть гипотезы. Зависит от количества графиков и сложности их построения. Минимум три графика различных типов.
Помимо графиков, построенных на семинаре, можно использовать идеи из галереи библиотеки matplotlib, seaborn.
