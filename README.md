## Цель проекта

Собрать собственный датасет (1000+ строк, 8+ признаков) и провести разведывательный анализ с визуализацией и формулировкой гипотез.
- Строк ≥1000 (для 4 чел.) - число фильмов
- Признаков ≥8–10
  - числовые (рейтинг, бюджет, сборы, длительность)
  - категориальные (жанры, страна производства)
  - текстовые (описание, теги)

## Признаки для анализа

| Признак | Описание |
|--------|----------|
| **year** | Год выхода фильма |
| **place** | Позиция фильма в списке того года / внутренний рейтинг выборки |
| **title** | Название фильма |
| **rating** | Рейтинг IMDb |
| **duration** | Длительность фильма в минутах |
| **director** | Режиссёр фильма |
| **genre** | Полный список жанров фильма (как на IMDb/Кинопоиске) |
| **release_date** | Дата премьеры фильма (YYYY-MM-DD) |
| **origin_country** | Страна производства фильма |
| **company** | Студия / производственная компания |
| **budget** | Бюджет фильма (USD) |
| **gross_us** | Кассовые сборы в США (USD) |
| **gross_world** | Мировые кассовые сборы (USD) |
| **open_week** | Сборы в первый уикенд (USD) |
| **release_month** | Месяц выхода фильма (1–12) |
| **genre_base** | Нормализованные базовые жанры для анализа |
| **has_budget** | Флаг наличия данных о бюджете |
| **has_gross_us** | Флаг наличия данных о сборах в США |
| **has_gross_world** | Флаг наличия данных о мировых сборах |
| **has_open_week** | Флаг наличия данных о сборах в первый уикенд |

## План анализа

## 1. Базовое исследование

### 1.1. Распределения
- Распределения ключевых признаков:
  - `rating`
  - `duration`
  - `budget`
  - `gross_world`

### 1.2. Сезонность релизов
- Когда выходят фильмы в течение года
- Пики перед праздниками  
*(гистограмма месяцев / countplot)*

### 1.3. Анализ жанров
- Топ популярных жанров
- Распределение длительности по жанрам  
*(boxplot / countplot)*


---

## 2. Бюджет → сборы → окупаемость

### 2.1. Связь бюджета и мировых сборов
- Scatterplot `budget → gross_world` (лог-лог)

### 2.2. Фильмы с мин. бюджетом и макс. сборами
- Выделить самые окупаемые фильмы (ROI → top)

### 2.3. Анализ ROI
- Топ-20 фильмов по ROI
- ROI по странам
- ROI по жанрам  
*(scatter, boxplot ROI по странам)*


---

## 3. Успешный по рейтингу, но провальный финансово

- Фильтры: высокий рейтинг + отрицательный ROI  
- Анализ:
  - жанры
  - релизные месяцы
  - студии / компании  
*(countplot жанров и месяцев)*


---

## 4. Жанры и успех

### 4.1. Влияние жанра на рейтинг и прибыль
- Boxplot рейтинга по жанрам
- Boxplot ROI по жанрам

### 4.2. География жанров
- Heatmap «страна × жанр»

### 4.3. Успешные комбинации жанров
- Генерация жанровых пар
- Barplot среднего ROI по жанровым комбинациям

### 4.4. Сравнение жанров
- Вывод: какие жанры поднимают прибыльность
- Какие жанры лучше по оценке зрителей

## 5. Собственная метрика успеха

### Расчёт
Формула (пример):
```bash
success_score = 0.4 * rating_norm + 0.4 * gross_norm + 0.2 * roi_norm
```

### Анализ
- Гистограмма `success_score`
- Топ-20 фильмов по метрике  
*(barplot)*


---

## 6. Самый успешный год в кино

### 6.1. Группировка по году
Считаем средние:
- `success_score`
- `rating`
- `roi`
- `gross_world`

### 6.2. Выводы
- Определить самый успешный год
- Какие жанры, компании и студии доминировали в этот год


---

## 7. Рецепт идеального фильма

### Итоговый профиль
- Оптимальная комбинация жанров
- Лучший сезон релиза
- Длительность успешных фильмов
- Диапазон бюджетов с максимальной окупаемостью
- Страны и студии, выпускающие «победителей»
- Режиссёры, чаще всего снимающие успешные фильмы

### Описательная формула
Сформировать текстовое правило вида:
> «Фильм длительностью X–Y минут, выпущенный в месяц Z, в жанровой комбинации A+B, с бюджетом B_range, созданный студией из страны C — имеет наибольший шанс стать успешным по метрике проекта.»
